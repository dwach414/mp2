Progress So Far

Here we have the current state of the game. You can view the sprite sheet or the game video by clicking on it. The sprite sheet is not in it's final form as these sprites were just borowed for development. We are in the process of hiring a new graphic designer in order to create our final sprites.

The game play itself is in early stages of development. We have only he basics of the game ready. Movement of the main character works as well as very simple enemy behaviours. These include following the player, attacking, and patrolling. As there are currently no obstacles on the board the enemy does no path-finding.
